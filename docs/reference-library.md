## 初步组合建议

本项目首版建议采用：

- `Ben Mingo / Adham Dannaway` 式的清晰首屏与信息层级；
- `Anton & Irene / Malika Favre` 式的强项目封面；
- `Rauno / Dennis Snellenberg` 式的精细反馈和页面过渡；
- 只保留一个明显的互动记忆点，避免视觉创意作品集变成交互技术演示。
