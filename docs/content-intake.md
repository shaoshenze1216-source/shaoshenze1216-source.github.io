# 作品集内容收集表

不需要一次写得很完整。先把已有信息放进来，后续再统一提炼和改写。

## 1. 个人定位

- 中文名 / 英文名：
- 希望使用的职业称谓：
- 一句话介绍：
- 所在城市 / 可工作地点：
- 目标受众：招聘方 / 客户 / 合作者 / 其他
- 希望访客看完后做什么：
- 邮箱：
- 社交链接：
- 简历文件：

## 2. 个人气质

- 希望别人记住你的三个关键词：
- 你最擅长解决的问题：
- 喜欢的网站或视觉参考：
- 明确不喜欢的风格：
- 喜欢的颜色 / 不喜欢的颜色：
- 动效偏好：克制 / 适中 / 强烈

## 3. 项目模板

为每个项目复制一份。

### 项目名称

- 一句话摘要：
- 时间：
- 项目类型：品牌 / 视觉 / 产品 / 活动 / 影像 / 其他
- 客户或所属团队：
- 我的角色：
- 我的具体职责：
- 背景与目标：
- 核心挑战：
- 关键思路与设计判断：
- 主要过程：
- 最终成果：
- 可量化结果或反馈：
- 合作者与署名要求：
- 是否可公开：
- 外部链接：
- 可用素材路径：封面、过程图、成品图、视频、品牌资产

## 4. 推荐素材规格

- 项目封面：横图优先，建议至少 1600 px 宽。
- 项目详情：同时准备横图、竖图和细节图，避免整页只有一种比例。
- 视频：优先 MP4/WebM，并提供静态封面。
- 头像：正方形或 4:5，两种裁切都可用。
- Logo / 图标：优先 SVG。
- 所有素材请标注版权、客户保密和可公开范围。
